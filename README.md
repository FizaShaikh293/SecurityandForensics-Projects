# 🔐 Cybersecurity & Digital Forensics Projects

This repository showcases practical and research-driven projects in cybersecurity and digital forensics, focusing on vulnerability analysis, attack simulation, and privacy-preserving blockchain investigation.

---

## 📌 Projects Included

### 1. Directory Traversal Attack Simulation

A hands-on security testing project demonstrating how improper input validation can lead to directory traversal vulnerabilities in web applications.

#### ~ Key Features:

* Exploitation of insecure file path handling
* Access to restricted system files (e.g., `/etc/passwd`)
* Security testing using Burp Suite and PortSwigger Labs
* Analysis of mitigation techniques such as input sanitisation and security headers

#### ~ Technologies Used:

* Burp Suite
* PortSwigger Web Security Labs

---

### 2. Privacy-Preserving Monero Transaction Analysis (Master’s Dissertation)

An advanced research project exploring how unsupervised machine learning can be used to analyse transaction behaviour on the Monero blockchain without compromising user anonymity.

#### ~ Key Features:

* Behavioural analysis using timing, frequency, and structural patterns
* Unsupervised anomaly detection using Isolation Forest and Autoencoders
* Dimensionality reduction using PCA
* Explainable AI using SHAP
* Fully integrated Python pipeline with Monero RPC
* Interactive visualisation dashboard using Streamlit

#### ~ Technologies Used:

* Python
* Monero RPC / monerod
* Scikit-learn
* TensorFlow / Keras
* SHAP
* Streamlit
